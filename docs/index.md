# DQX - Data Quality Excellence

DQX is a Python 3.11+ library for data quality validation that lets you write validation logic as testable Python functions executing efficiently on SQL backends (DuckDB, BigQuery, Snowflake).

## What's here

- **[Data Contracts](design/contracts/index.md)** — Technical specification for defining and enforcing data quality contracts, including check types, SLA definitions, and the type system.
- **[Profiles](design/misc/profiles/index.md)** — Configurable data quality rules that adapt to the shape and history of your data.
